# IDEAS

本文件用于记录新想法，但不代表立即开发。

## 当前暂存想法

### 1. Replay Engine

读取历史订单、聊天和售后记录，让 AI 重新执行，并与人工结果对比。

目标指标：

- 准确率
- 满意度
- 响应时间
- 人工接管率

### 2. Simulation Engine

模拟客户咨询、砍价、退款、投诉、物流和售后场景。

长期目标：每天模拟 10000+ 客户。

### 3. Evaluation Engine

为每个 Agent 建立评分体系。

候选工具：

- DeepEval
- Promptfoo
- Ragas
- Langfuse

### 4. Training Center

老板和客服修改 AI 回复后，进入训练中心。

训练中心负责把修改沉淀到：

- Memory
- Knowledge
- Workflow
- Prompt

### 5. AI运营私域获客

识别高意向客户，生成私域引导话术和投流计划草稿。

注意：预算、群发、广告投放必须老板审批。
